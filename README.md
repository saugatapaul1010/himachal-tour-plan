# Himachal Tour Plan

Interactive photography trip planner for Himachal, Oct 2026 — route map, day flow, itinerary, live GPS, checklist, budget.

**Live site:** https://saugatapaul1010.github.io/himachal-tour-plan/

## About

This is a single self-contained static HTML file (`index.html`). There is no build step, no server, and no environment variables — open the file in any modern browser and it works. All assets are inlined; external data is fetched at runtime from public APIs.

The **Live GPS** tab requires location permission and works best on a phone.
